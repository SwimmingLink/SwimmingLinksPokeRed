# Pokémon Red and Blue [![Build Status][ci-badge]][ci]

This is a modified disassembly of Pokémon Red and Blue.

It builds the following ROMs:

- pokered.gbc
- pokeblue.gbc
- pokeblue_debug.gbc

To set up the repository, see [**INSTALL.md**](INSTALL.md).
